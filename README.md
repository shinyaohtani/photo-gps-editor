# PhotoGPSEditor

PhotoGPSEditor は、GPS を持たない Canon 写真に対して、iPhone 側で記録した位置情報付き写真の時刻と場所を基準に GPS を補間し、必要なら地図上で微修正してから EXIF に保存するための macOS アプリです。

このアプリは、たとえば旅行中に iPhone では位置情報付きで撮れているが、Canon 側の写真には GPS が入っていない、というときに使います。まず iPhone 側の JSON を読み込み、次に Canon 側の写真フォルダを読み込み、撮影時刻をもとに Canon 写真へ位置を推定で割り当てます。そのあと地図上でズレを見ながら少しずつ直し、最後に Canon 写真ファイルへ GPS を書き戻します。

## 事前に用意するもの

- iPhone 側の位置情報付き写真 JSON
- Canon 側の写真または動画が入ったフォルダ
- `exiftool`
  - このアプリは撮影時刻の読み出しと EXIF 保存に `exiftool` を使います
  - 例: `brew install exiftool`

## このアプリでできること

- iPhone 側の参照写真を地図上に読み込む
- Canon 側の写真を撮影時刻つきで読み込む
- 時刻ベースで Canon 写真へ GPS を自動補間する
- 地図上で Canon 写真の位置をまとめて微修正する
- 修正後の GPS を Canon 写真ファイルへ保存する

## 画面の見方

- 黒い小さな点
  - iPhone 側の参照写真です
  - 基準点として表示されます
  - ドラッグ移動はできません
- オレンジまたは緑の点
  - Canon 側の対象写真です
  - こちらを補間し、必要なら手で微修正します
- 左のリスト
  - Canon 写真一覧、または選択中の写真一覧を確認できます
- 上部ツールバー
  - `Load JSON`
  - `Load Canon`
  - `TZ`
  - `Interpolate`
  - `Save EXIF`

## 最初から最後までの操作順

### 1. アプリを起動する

アプリを起動すると、前回の固定パスが存在する場合は自動で読み込まれることがあります。意図したデータでない場合は、あとで `Load JSON` と `Load Canon` で読み直してください。

### 2. iPhone 側の JSON を読み込む

上部の `Load JSON` を押し、iPhone 側の位置情報付き写真 JSON を選びます。

ここで地図上に黒い小さな点が表示されます。これらは参照用であり、編集対象ではありません。

### 3. Canon 側の写真フォルダを読み込む

上部の `Load Canon` を押し、GPS を付けたい Canon 写真フォルダを選びます。

読み込み後、Canon 側の写真が一覧に出ます。まだ GPS が無いものは `No GPS` と表示されます。

### 4. タイムゾーンを確認する

上部の `TZ` で、Canon 写真の撮影時刻をどのタイムゾーンとして解釈するかを確認します。

- 日本で撮った写真なら `JST +9`
- イタリア旅行なら通常は `CET +1`
- 撮影機材の時刻設定が UTC なら `UTC`

ここがずれていると補間結果が全体的にずれます。海外旅行の写真では特に先に確認してください。

### 5. GPS を自動補間する

`Interpolate` を押すと、iPhone 側の位置記録を基準に Canon 写真へ GPS が自動で割り当てられます。

この時点で、Canon 側の点が地図上に並びます。完全に正確でなくても問題ありません。まずは大まかに合っているかを見ます。

### 6. 補間結果を確認する

地図を見ながら、Canon 側の点が旅行ルートや撮影場所と大きく外れていないか確認します。

確認のコツ:

- 参照用の黒点の流れと、Canon 側の点の流れがだいたい一致しているか見る
- 一部だけずれているのか、全体が時間方向にずれているのかを見る
- 全体がずれているなら、まず `TZ` を疑う

### 7. 写真を見ながら位置を確認する

地図上の Canon 側の点にマウスを乗せると、対応する写真が大きく表示されます。

- 右キーまたは下キーで次の時刻の写真へ進めます
- `Esc` で拡大表示を閉じます
- マウスが画像や元の点から離れると表示は閉じます

写真を見ながら、建物や広場、道路、海岸線などを目印にして、どこへ置くべきか判断してください。

### 8. 微修正したい写真を選ぶ

選び方は複数あります。

- 左のリストをクリックする
- 地図上の点をクリックする
- 地図上でドラッグして矩形選択する
- `Shift` を押しながらクリックして選択を追加または解除する
- 地図上で右クリックして、その場所に重なっている写真一覧から選ぶ

写真が密集している場所では、まず矩形選択である程度まとめて選ぶのが扱いやすいです。

### 9. 地図上で位置を微修正する

Canon 側の写真を移動するときは、`Option` を押しながら地図上でドラッグします。

通常のドラッグは矩形選択です。移動ではありません。誤って位置を動かさないため、移動操作は `Option + Drag` に分かれています。

微修正の基本:

- まず近い時刻の写真を複数まとめて選ぶ
- `Option + Drag` で少しだけ動かす
- 写真プレビューを見て合っているか確認する
- 必要ならさらに少し動かす

### 10. 必要ならやり直す

位置を動かしすぎた場合や、補間からやり直したい場合は macOS 標準の Undo が使えます。

- `Command + Z` で元に戻す
- `Command + Shift + Z` でやり直す

### 11. 保存前に対象を再確認する

左のリスト上部で `Selected` や `No GPS` を使うと、確認したい対象だけに絞れます。

保存前には次を確認してください。

- Canon 側で保存したい写真に GPS が入っているか
- 明らかに海の上や別都市へ飛んでいる写真が残っていないか
- タイムゾーンの設定ミスがないか

### 12. EXIF へ保存する

問題なければ `Save EXIF` を押します。

この操作で、Canon 側の写真ファイルへ GPS が書き込まれます。保存対象は GPS を持つ Canon 写真です。参照用の iPhone JSON は書き換えません。

### 13. 終了する

保存後、必要なら Finder や他の写真管理ソフトで EXIF を確認し、問題なければアプリを終了してください。

## 地図操作の要点

- クリック: 1 枚選択
- `Shift + Click`: 選択追加または解除
- ドラッグ: 矩形選択
- `Option + Drag`: 選択中の Canon 写真を移動
- 右クリック: 近い場所にある複数写真の選択リストを開く
- `Command + Z`: 元に戻す
- `Command + Shift + Z`: やり直し
- `Esc`: 写真プレビューを閉じる

## こんなときはここを確認

### 補間位置が全体的にずれている

`TZ` が正しいか確認してください。旅行写真で最も起きやすい原因です。

### 地図上に黒い点しか出ない

Canon フォルダが未読込か、まだ `Interpolate` を実行していない可能性があります。

### `Save EXIF` しても反映されない

`exiftool` が入っているか、対象ファイルに書き込み権限があるか確認してください。

### 参照点を動かしたい

参照点は基準データなので動かせません。動かすのは Canon 側の点だけです。
